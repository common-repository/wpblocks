A collection of additional, beautifully designed blocks for Gutenberg editor.

== Description ==

WPBlocks is a free plugin that adds additional blocks to the Gutenberg editor. Beautifully designed blocks integrate seamlessly with the new WordPress editor.

== Available Blocks ==

Currently, there are three additional blocks available with more coming with plugin updates.

* **Info Box** - Displays beautiful notification boxes in 4 styles.
* **Social Icons** - Displays links to your social profiles.
* **Empty Space** - Adds empty spacer to separate blocks.

== Installation ==

= Automatic Installation =

Automatic installation is the easiest way to install WPBlocks as WordPress handles the file transfers itself. To do an automatic install of 
WBlocks, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “WPBlocks” and click Search Plugins. Once you’ve found our plugin simply click “Install Now”. After the installation has finished, click the “Activate Plugin“ button.

WPBlocks requires Gutenberg, so make sure you have installed and activated the [Gutenberg plugin](https://wordpress.org/plugins/gutenberg/).

= Manual Installation =

1. Upload the `wpblocks` folder to your `/wp-content/plugins` directory or alternatively upload the `wpblocks.zip` file via the \'Add New\' button in WordPress plugins page.
2. WPBlocks requires Gutenberg plugin. Install and activate Gutenberg WordPress plugin.
3. Activate WPBlocks through the \'Plugins\' menu in WordPress.

== Frequently Asked Questions ==

= What is Gutenberg? =

Gutenberg is a new WordPress editor that is going to be merged with WordPress 5.0.

It\'s important to keep in mind that Gutenberg is still in beta version and not yet fully functional, feature complete, or production ready. Once ready, it\'ll come with every WordPress installation (not as a plugin, but built-in as the default editor).

= I installed this plugin but don\'t see any additional blocks. =

Each block can be enabled/disabled from the “WPBlocks” panel under “Settings” menu item in WordPress Dashboard. Check if the blocks you want to use are enabled.

= What are the additional blocks added by this plugin? =

Currently, there are three additional blocks available: Info Box, Social Icons, and Blank Space. More blocks will be added with updates.

= Is WPBlocks translation ready? =

Yes. WPBlocks comes with full translation and localization support via the wpblocks textdomain. All .mo and .po translation files should go into the languages folder in the root folder of the plugin.
